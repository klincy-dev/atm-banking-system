# ATM Banking System (Python)

A secure ATM simulation system built using Python.

## Features
- Multi-user ATM system
- Auto account number generation
- Secure PIN hashing (SHA256)
- Daily withdrawal limit
- Account lock after 3 failed attempts
- Transaction receipt generator
- Admin panel to unlock accounts

## Technologies
- Python
- JSON Database

## Run the Project

python atm.py
